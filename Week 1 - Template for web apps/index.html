Tasks

To make sure you've set up your environment correctly, complete the following tasks:

  1. Install Node.js
  2. Install and configure Firefox Developer Edition
    1. Install AXE extension
  3. Install VS Code
    1. Install Debugger for Firefox extension
    2. Install JSLint extension
    3. Install Mocha Test Explorer extension
  4. GitHub
    1. Create your own GitHub Account
    2. Accept the GitHub Classroom coursework assignment.
    3. Find the fourier-space / Computing-2-Applications repo
  5. Set Up VSCode
    1. Create a new folder for Computing 2
    2. Clone the module repo into your workspace
    3. Clone your assignment repo into your workspace
  6. Check you can run the module sample web app.
    1. Can you see this webpage!
  7. Edit the README.md file in your submission repo with your CID and push.
